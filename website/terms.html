【风险特别提示】

证券及其衍生品交易具有极高的风险。QuantMind 提供的所有功能（包括 AI 策略生成、回测引擎、因子库）仅作为技术工具使用，不构成任何投资建议。您应拥有独立的投资判断能力，并自行承担因使用本平台而产生的所有交易风险与损失。

一、协议的确认与接受

本《用户服务协议》（以下简称"本协议"）是您（以下称"用户"）与 QuantMind（以下称"我们"）之间就 QuantMind 量化交易平台（以下简称"本平台"）所提供的产品及服务所订立的法律协议。

当您按照页面提示完成注册流程、或者实际使用本平台服务时，即视为您已充分阅读、理解并接受本协议的全部内容。 如果您不同意本协议或其中任何条款，请立即停止注册或使用流程。

二、服务内容与许可

QuantMind 为用户提供包括但不限于以下技术支持：

  • 开源版 (Open Source)：基于 Python 的量化回测引擎及 IDE 环境。
  • 社区版 (Community)：包含云端数据同步、AI 策略助手及高频回测能力。
  • 企业版 (Enterprise)：私有化部署、极速柜台接入及多租户策略管理系统。

我们授予您一项个人的、不可转让的、非排他性的软件使用许可，以使用本平台提供的各项工具。

三、量化交易风险披露

您在使用本平台进行量化研究与执行时，必须明确知晓并接受以下风险：

  1. 过往业绩不代表未来表现：回测（Backtest）是在历史数据上的模拟运行，其收益曲线无法保证在实盘环境中复现。
  2. 策略失效风险：市场环境波动、制度更新或流动性崩塌可能导致原有的量化模型瞬间失效。
  3. 技术与网络风险：因云端服务器延迟、网络中断或券商柜台（QMT/Ptrade）接口故障导致的下单失败、滑点过大或交易异常。
  4. 模型偏差风险：AI 辅助生成的策略逻辑可能存在隐含的过拟合或逻辑漏洞，需由用户自行完成严格审计。

四、知识产权与资产所有权

  1. 平台所有权：QuantMind 及其关联软件的底层架构、源代码（不含明确开源部分）、数据字典、UI 设计及"AI 独家因子"的知识产权均归我们所有。
  2. 用户策略所有权：用户利用本平台工具编写的量化策略逻辑、私有模型参数及个性化特征工程，其知识产权及所有权完全归用户所有。
  3. 保留权利：我们保留对恶意破解、逆向工程、反编译本平台核心加密模块进行追究法律责任的权利。

五、用户行为规范

您承诺在使用本平台服务时，严禁从事以下活动：

  • 利用本平台进行任何形式的市场操纵（如对倒、自买自卖）或其他违反证监会监管规定的交易行为。
  • 利用本平台漏洞进行恶意套利、非法爬取数据或干扰其他用户策略的运行。
  • 向本平台上传病毒、恶意脚本或试图攻击 QuantMind 服务器。
  • 未经书面许可，将本平台的因子数据通过 API 二次转售给第三方牟利。
