QuantMind

AI 驱动的量化交易平台

QuantMind 是一个开源的一站式量化交易解决方案，集模型训练、回测分析、实盘交易于一体，让量化交易从未如此简单。

快速开始 查看源码

核心特性

数据平台

自研 A 股数据中心，146 维特征因子库，支持 T-0 实时行情推送

模型训练

集成 Qlib 与 LightGBM/LSTM，支持分布式训练与自动超参调优

回测引擎

高并发历史仿真，内置风控验证与全景可视化分析

实盘交易

QMT 网关原生对接，毫秒级信号执行，支持极速柜台

集群管理

Kubernetes 原生调度，支持弹性伸缩与多租户隔离

开源免费

AGPL-3.0 协议开源，无功能限制，支持私有化部署

快速体验

在 Ubuntu 22.04+ 服务器上执行一行命令，即可完成部署：

bash 
# 一键部署 QuantMind
curl -fsSL https://gitee.com/qusong0627/quantmind/raw/master/deploy/quick-deploy.sh | sudo bash -s -- --yes

部署完成后访问 http://<服务器IP>，默认账号 admin / admin123。 更多详情请查看 快速开始 文档。

系统架构

QuantMind 采用微服务架构设计，各模块可独立部署或作为单体应用运行：

模块 说明 技术栈
quantmind-engine 核心引擎服务，处理训练、回测、推理任务 Python / Qlib / FastAPI
quantmind-trade 交易网关，对接 QMT 实盘通道 Python / QMT SDK
quantmind-api API 网关，统一认证与流量控制 FastAPI / PostgreSQL
quantmind-stream 实时行情推送，WebSocket 服务 FastAPI / WebSocket
