什么是 QMT Agent？

QMT Agent 是 QuantMind 平台的实盘交易桥接代理程序，运行在您的 Windows 电脑上，作为本地 QMT 交易终端与云端 QuantMind 平台之间的安全桥梁。

通过 QMT Agent，您可以将 QuantMind 策略产生的交易信号自动转发到 QMT 券商接口，实现真正的自动化实盘交易。

核心功能

安全桥接

使用 Access Key + Secret Key 鉴权，短期 Token 自动刷新，WebSocket 加密传输，确保交易指令安全可靠。

实时心跳

周期性心跳上报与账户快照同步，云端实时监控 Agent 状态，异常断线自动告警。

智能下单

支持同步/异步下单、保护限价单、智能执行重试，大批量委托自动排队派发，避免柜台拥塞。

自动重连

QMT 断线自动重连，WebSocket 连接异常自动恢复，启动时补偿查询历史委托，降低断线窗口状态缺失风险。

桌面 GUI

现代化桌面界面，可视化配置管理，实时状态监控，托盘常驻运行，崩溃自动重启。

融券支持

支持信用账户融券做空，自动校验可融额度，做空准入前置检查，标准错误码反馈。

快速开始

1. 下载并安装

从 下载页面 获取 QMT Agent 安装包，安装到您的 Windows 电脑。

2. 获取凭证

登录 QuantMind 平台，进入「实盘交易」→「QMT 连接中心」，创建或复制您的 Access Key 和 Secret Key。

3. 配置 Agent

启动 QMT Agent 桌面程序，填写 Access Key、Secret Key、资金账号，并指定 QMT 安装路径。

4. 启动连接

确保 QMT 已启动并登录，点击「启动 Agent」按钮，状态显示「在线」即表示连接成功。

5. 开始交易

在 QuantMind 平台创建实盘策略，交易信号将自动通过 Agent 转发到 QMT 执行。

系统要求

  • 操作系统：Windows 10 / 11（64位）
  • QMT 版本：迅投 QMT 极简版（MiniQMT）
  • 网络：稳定的互联网连接，需能访问 QuantMind 云端服务
  • 权限：需要读取 QMT 数据目录的权限

常见问题

Q: QMT Agent 是否安全？

A: 是的。Agent 仅作为桥接代理，不会存储您的券商密码。所有通信使用加密通道，凭证使用短期 Token 机制，支持本地 API 鉴权。

Q: Agent 需要一直运行吗？

A: 是的。Agent 需要在交易时段保持运行，才能接收云端交易指令并转发到 QMT。建议使用托盘模式后台运行。

Q: 支持哪些券商？

A: 目前支持迅投 QMT 接入的券商，包括多家主流券商的量化交易通道。具体请咨询您的券商是否支持 QMT。

Q: 如何排查连接问题？

A: Agent 提供详细的运行日志，可通过桌面界面查看或导出诊断包。常见问题包括：QMT 未启动、路径配置错误、凭证过期等。
