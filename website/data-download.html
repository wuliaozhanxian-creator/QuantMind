安装说明

下载数据包

从中国移动云盘下载数据压缩包。

解压并替换目录

解压后替换项目根目录下的 db 和 models 目录：

目录结构
quantmind/
├── db/                  # Qlib 数据目录
│   ├── qlib_data/       # 股票特征数据
│   │   ├── calendars/
│   │   ├── instruments/
│   │   └── features/
│   └── feature_snapshots/
├── models/              # 预训练模型
│   └── model_preview/
├── backend/
└── ...

验证数据

运行以下命令验证数据是否正确加载：

python
import qlib
from qlib.data import D

# 初始化 Qlib
qlib.init(provider_uri="./db/qlib_data")

# 测试数据加载
df = D.features(["SH600519"], ["$close", "$volume"])
print(df.head())

数据更新

  • 更新频率：每周周末更新
