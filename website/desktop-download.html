macOS 安装指南

由于 macOS 安全机制限制，首次运行可能遇到"无法验证开发者"或"应用已损坏，打不开"的提示。请按以下步骤操作：

方法一：右键打开（解决“无法验证开发者”）

  1. 下载 .dmg 安装包并双击打开
  2. 将 QuantMind量化助手 拖拽到 Applications (应用程序) 文件夹
  3. 在 Finder (访达) 中找到该应用
  4. 右键点击应用图标，选择"打开" (Open)
  5. 在弹出的对话框中点击"打开"确认

方法二：命令行绕过（解决“应用已损坏，打不开”）

如果双击或右键打开仍提示“应用已损坏”，这是因为 macOS 对下载的文件进行了隔离，请在终端执行以下命令：

Terminal
# 移除隔离属性（请确保已将应用拖入应用程序文件夹）
sudo xattr -rd com.apple.quarantine /Applications/QuantMind量化助手.app

* 执行后可能需要输入您的开机密码（输入时密码不显示），完成后即可正常双击打开。

为什么会出现这些提示？

macOS 默认只允许运行来自 App Store 或已签名开发者的应用。QuantMind 桌面客户端目前处于预览阶段，未经过 Apple 付费公证，因此需要手动授权。这不影响应用的安全性和功能。
