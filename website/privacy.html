隐私政策

版本：QM-PRIVACY-2026-FINAL | 更新日期：2026 年 04 月 21 日

QuantMind（以下简称"我们"或"QuantMind"）深知个人信息对您的重要性，我们将按照法律法规的规定，保护您的个人信息及隐私安全。我们制定本隐私政策并特别提示：希望您在使用 QuantMind 服务前仔细阅读并理解本政策，以便您做出适当的选择。

前言

本政策适用于 QuantMind 运行的所有产品及服务，包括但不限于 QuantMind 官网、桌面客户端（Windows/macOS）、量化引擎（Engine）、数据中心（Stream）以及相关 API 调用服务。我们严格遵守《中华人民共和国个人信息保护法》、《中华人民共和国网络安全法》以及《中华人民共和国数据安全法》等相关法律法规。

一、我们如何收集和使用您的个人信息

个人信息是指以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息。我们会出于以下目的，收集和使用您的个人信息：

1. 账户注册与身份验证

  • 基础服务：当您注册 QuantMind 账户时，您需要提供手机号码或电子邮箱地址。这些信息是为您提供量化计算、策略托管及同步服务的必要基础。
  • 安全性增强：为了保障账户安全，我们会收集您的登录日志，包括登录时间、地理位置（基于 IP）、登录方式。

2. 订阅服务与订单处理

当您购买 QuantMind 的增值服务时，我们会收集您的交易记录，包括订单号、支付时间、订单金额、支付渠道。请注意，我们不会收集或存储您的银行卡卡号、支付密码或 CVV 码等高度敏感信息。

3. 软件运行与故障排查

为了保障软件的正常运行，并对系统崩溃或性能瓶颈进行分析，我们会收集以下信息：

  • 设备标识：UUID、硬件型号、操作系统及版本、语言设置、分辨率。
  • 性能日志：因子计算耗时、回测引擎 CPU/内存占用、API 调用成功率。

二、量化资产所有权与隐私保护专项

作为金融量化工具，我们深知策略（Weights & Alpha）是您的核心竞争力。QuantMind 特此做出以下独立承诺：

  1. 策略逻辑所有权：您编写的所有脚本、量化模型、参数配置、交易信号的所有权完全且绝对地归属于您个人。QuantMind 后端不会以任何形式自动读取或上传您的本地代码，除非您显式触发"同步/备份"功能。
  2. API 密钥安全：您的柜台账号（如 QMT/Ptrade）、私有 API 密钥采用 AES-256 加密算法进行本地化存储。这些敏感密钥永远不会被传输至 QuantMind 服务器。
  3. 训练环境隐私：在使用 AI 模型自动辅助生成功能时，系统仅传输您主动输入的自然语言指令，不会擅自读取您的本地数据文件作为训练材料。

三、信息的存储与保护措施

1. 存储期限

我们只会在达成本政策所述目的所需的期限内保留您的个人信息，除非法律有强制的存留要求，例如《中华人民共和国网络安全法》要求留存网络日志不少于六个月。

2. 技术安全

  • 全站加密：采用符合行业标准的 TLS 1.3 传输加密协议。
  • 脱敏处理：对于展示在前端页面的敏感信息（如手机号、邮箱），我们进行掩码处理（去标识化）。
  • 隔离审计：生产环境数据库采用严格的角色访问控制（RBAC），所有运维操作均有完整审计日志。

五、您对个人信息的控制权

根据《中华人民共和国个人信息保护法》，您对您的个人信息享有以下权利：

  • 更正权：您有权在 QuantMind 设置中心更新您的个人资料。
  • 注销权：您可以申请注销账户。注销后，我们将立即停止为您提供产品或服务，并根据您的要求删除您的个人信息，法律法规另有规定的除外。
  • 撤回授权：您可以点击"隐私设置"关闭同步功能，即可撤回对云端备份数据的处理授权。

六、未成年人保护

QuantMind 的产品主要面向具有民事行为能力的量化投资者和专业金融从业者。我们不向未满 18 周岁的未成年人提供服务。如果我们发现无意中收集了未成年人的个人信息，我们会立即设法删除相关数据。

七、政策更新与通知

QuantMind 保留适时更新本政策的权利。未经您明确同意，我们不会削减您按照本隐私政策所应享有的权利。我们会通过显著位置公告或推送通知向您展示本政策的变更。若您继续使用我们的服务，即表示您同意受经修订的本政策约束。

如果您对本政策有任何疑问、投诉或建议，请通过以下方式联系：
