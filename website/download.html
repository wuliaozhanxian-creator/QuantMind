下载安装

获取 QuantMind 最新版本，开始您的量化之旅。

系统要求

配置项 最低要求 说明
Python 版本 3.10 必须使用 Python 3.10，其他版本不兼容
内存（基础服务） ≥ 8GB 运行回测、API 服务、数据下载等基础功能
内存（模型训练/推理） ≥ 64GB 模型训练、批量推理、大规模回测所需
磁盘空间 ≥ 50GB 历史数据、因子库、模型文件存储
操作系统 Windows 10+ / macOS 11+ / Ubuntu 20.04+ 推荐 Linux 服务器部署
重要提示

Python 版本必须为 3.10，使用 3.11/3.12 可能导致依赖安装失败或运行时错误。

下载方式

方式一：一键部署（推荐）

在 Ubuntu 22.04+ 服务器上执行以下命令，自动完成全部部署：

bash
# 一键部署（自动安装 Docker、数据库、后端、前端）
curl -fsSL https://gitee.com/qusong0627/quantmind/raw/master/deploy/quick-deploy.sh | sudo bash -s -- --yes

部署完成后访问：http://<服务器IP>，默认账号：admin / admin123

方式二：源码安装

从 Gitee 克隆源码，适合开发者自定义修改：

bash
git clone https://gitee.com/qusong0627/quantmind.git
cd quantmind

# 安装依赖
pip install -r requirements.txt

# 启动服务
docker-compose up -d

方式三：Docker Compose

已有 Docker 环境的用户可直接使用：

bash
git clone https://gitee.com/qusong0627/quantmind.git
cd quantmind

# 构建并启动所有服务
docker compose up -d --build
