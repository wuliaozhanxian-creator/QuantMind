> **数据源**：fundamental_aligned.parquet（89维，7.9M行，2020-01~2026-04，唯一事实源）

## 二、 实盘筛选策略模板

### 2.1 趋势突破策略

```sql
-- 盘前筛选：均线多头排列 + 偏离度合理
SELECT symbol, stock_name, close, ma5, ma10, ma20, ma60
FROM stock_daily_latest
WHERE trade_date = CURRENT_DATE - 1
  -- 多头排列
  AND ma5 > ma10 AND ma10 > ma20 AND ma20 > ma60
  -- 收盘价在5日线上方但不偏离过大
  AND ma_gap_5 BETWEEN 0 AND 0.05
  -- 排除ST和次新股
  AND is_st = 0 AND listed_days > 60
  -- 市值过滤（10亿-500亿）
  AND total_mv BETWEEN 1e9 AND 5e10
  -- 流动性过滤
  AND volume_ma_5 > 10000
```

**盘中确认条件**：
- `volume_ratio_5 > 1.5`（放量突破）
- `pct_change > 0.02`（涨幅确认）
- `limit_up_today = 0`（未涨停，可买入）

### 2.2 超跌反弹策略

```sql
-- 盘前筛选：RSI超卖 + 均线偏离
SELECT symbol, stock_name, close, rsi_6, rsi_14, ma_gap_20, return_10d
FROM stock_daily_latest
WHERE trade_date = CURRENT_DATE - 1
  -- RSI超卖
  AND rsi_6 < 30 AND rsi_14 < 35
  -- 偏离20日线较远
  AND ma_gap_20 < -0.1
  -- 近期跌幅较大
  AND return_10d < -0.1
  -- 排除ST
  AND is_st = 0
  -- 非退市风险
  AND listed_days > 180
```

**盘中确认条件**：
- `pct_change > 0`（当日翻红）
- `volume_ratio_5 > 1.2`（放量企稳）

### 2.3 资金驱动策略

```sql
-- 盘前筛选：流通比例合理 + 利润增长
SELECT symbol, stock_name, inst_ownership, profit_growth, roe, pe_ttm
FROM stock_daily_latest
WHERE trade_date = CURRENT_DATE - 1
  -- 流通比例大于30%（非高度控股）
  AND inst_ownership > 0.3
  -- 利润正增长
  AND profit_growth > 0.15
  -- ROE优良
  AND roe > 10
  -- 估值合理
  AND pe_ttm BETWEEN 10 AND 40
  -- 沪深300成分
  AND idx_hs300 = 1
```

**盘中确认条件**：
- `main_flow > 0`（主力净流入）
- `b_volume > s_volume`（外盘大于内盘）
- `volume_ratio_5 > 1`（温和放量）

### 2.4 成长股策略

```sql
-- 盘前筛选：高成长 + 合理估值
SELECT symbol, stock_name, pe_ttm, profit_growth, roe, label
FROM stock_daily_latest
WHERE trade_date = CURRENT_DATE - 1
  -- 标签为成长或价值成长
  AND label IN (2, 3)
  -- 利润增长>30%
  AND profit_growth > 0.30
  -- ROE>5%
  AND roe > 5
  -- PE<50（避免极端值）
  AND pe_ttm BETWEEN 0 AND 50
```

### 2.5 打板策略

```sql
-- 盘前筛选：连板潜力股
SELECT symbol, stock_name, consecutive_limit_up_days, total_mv, turnover_rate
FROM stock_daily_latest
WHERE trade_date = CURRENT_DATE - 1
  -- 已有连板
  AND consecutive_limit_up_days >= 1
  -- 市值适中（方便拉升）
  AND total_mv BETWEEN 2e9 AND 1e10
  -- 换手率适中
  AND turnover_rate BETWEEN 5 AND 20
  -- 非ST
  AND is_st = 0
```

**盘中确认条件**：
- `limit_up_today = 1`（已涨停）
- `volume_ratio_5 < 0.5`（缩量涨停，封板牢固）

## 三、 因子分类速查表

### 3.1 趋势类因子

| 因子 | 用途 | 多头信号 | 空头信号 |
|------|------|----------|----------|
| ma_gap_5 | 短期偏离 | > 0 | < -0.03 |
| ma_gap_20 | 中期偏离 | > 0 | < -0.05 |
| macd_dif | MACD快线 | > 0 且上升 | < 0 且下降 |
| macd_hist | MACD柱状图 | > 0 | < 0 |

### 3.2 动量类因子

| 因子 | 用途 | 强势信号 | 弱势信号 |
|------|------|----------|----------|
| return_5d | 5日动量 | > 0.05 | < -0.05 |
| return_20d | 20日动量 | > 0.1 | < -0.1 |
| rsi_6 | 短期强弱 | 50-80 | < 30 |
| kdj_j | KDJ敏感线 | > 50 | < 20 |

### 3.3 估值类因子

| 因子 | 用途 | 低估信号 | 高估信号 |
|------|------|----------|----------|
| pe_ttm | 市盈率 | < 15 | > 50 |
| pb | 市净率 | < 1.5 | > 5 |
| bp | 账面市值比 | > 0.5 | < 0.2 |
| ep_ttm | 盈利收益率 | > 0.06 | < 0.02 |

### 3.4 波动类因子

| 因子 | 用途 | 低波信号 | 高波信号 |
|------|------|----------|----------|
| vol_std_20 | 20日波动率 | < 0.02 | > 0.05 |
| vol_atr_14 | ATR波动 | 小 | 大 |
| beta_20 | 相对大盘弹性 | < 0.8（防御） | > 1.2（进攻） |

### 3.5 资金类因子

| 因子 | 用途 | 流入信号 | 流出信号 |
|------|------|----------|----------|
| main_flow | 主力净流入 | > 0 | < 0 |
| flow_net_amount | 资金净流入 | > 0 | < 0 |
| b_volume vs s_volume | 内外盘对比 | b > s | s > b |
| inst_ownership | 流通比例 | > 30% | < 10% |

### 3.6 分类标签

| 标签值 | 含义 | 定义 |
|--------|------|------|
| 0 | 周期/其他 | 不满足价值/成长条件 |
| 1 | 价值 | pe_ttm<30, pb<2, roe>2 |
| 2 | 成长 | profit_growth>0.3, roe>2 |
| 3 | 价值成长 | 同时满足价值和成长条件 |

> 注：`label` 基于 pe_ttm、pb、roe、profit_growth 自动分类，覆盖率 98.1%。

## 四、 实盘工作流

### 4.1 盘前（9:00-9:25）

```
1. 加载 T-1 日 stock_daily_latest 数据
2. 执行盘前筛选 SQL（按策略模板）
3. 生成候选股票池（建议 20-50 只）
4. 设置盘中监控条件
```

### 4.2 盘中（9:30-15:00）

```
1. 实时获取候选股行情
2. 计算盘中实时因子（volume_ratio, pct_change 等）
3. 触发买入/卖出信号
4. 风控检查（涨跌停、流动性）
```

### 4.3 盘后（15:00-16:00）

```
1. 更新 stock_daily_latest 全量数据
2. 回测当日信号准确率
3. 调整次日筛选参数
```

## 五、 风控规则

| 规则 | 条件 | 动作 |
|------|------|------|
| ST过滤 | is_st = 1 | 排除 |
| 次新过滤 | listed_days < 60 | 排除（可选） |
| 涨停不可买 | limit_up_today = 1 | 禁止买入 |
| 跌停不可卖 | limit_down_today = 1 | 禁止卖出 |
| 流动性检查 | volume_ma_5 < 5000 | 排除 |
| 市值下限 | total_mv < 1e9 | 排除 |
| 概念过滤 | concept_* = 0（全部） | 至少命中1个热点概念 |
| PE异常 | pe_ttm IS NULL OR pe_ttm > 500 | 排除或子策略 |

## 六、 因子相关性提示

高相关因子组（避免重复使用）：

| 因子组 | 相关系数 | 建议 |
|--------|----------|------|
| return_5d / ma_gap_5 | > 0.8 | 二选一 |
| rsi_6 / kdj_j | > 0.7 | 二选一 |
| volume_ratio_5 / volume_ma_5 | > 0.6 | 配合使用 |
| pe_ttm / ep_ttm | = -1.0 | 互为倒数，选一 |
| total_mv / ln_mv_total | > 0.95 | 选一（ln去偏更稳定） |
| b_volume / lrg_trd_tolbuynum | > 0.7 | b_volume 覆盖率更高 |
| main_flow / flow_net_amount | > 0.9 | main_flow 优先 |

## 七、 字段覆盖率说明

| 覆盖率 | 字段数 | 说明 |
|--------|--------|------|
| 100% | 30 | 概念板块、指数成分、涨跌停、基础信息等 |
| ≥95% | 46 | 行情、技术指标、均线、波动率、估值（除pe_ttm） |
| 91-93% | 11 | b/s_volume, micro_*, beta_20, main_flow, flow_net_amount, consecutive_limit_up_days, lrg_trd_* |
| 74% | 2 | pe_ttm, ep_ttm（亏损股无PE，基本面限制） |

策略使用建议：
- 使用 `pe_ttm/ep_ttm` 时加 `IS NOT NULL` 或 `> 0` 过滤
- 使用 91-93% 字段时加 `dropna()` 确保数据完整性
