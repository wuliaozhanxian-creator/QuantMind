# styles

用途：样式与主题资源。

## 说明
- 归属路径：electron\src\styles
- 修改本目录代码后请同步更新本 README
- 全平台统一圆角样式：启动时为 `html` 注入 `qm-rounded`，仅对 `body` 应用 `border-radius: 16px` + `overflow: hidden`，简化层级避免重复规则。
- Windows 额外适配：当 `html` 存在 `qm-win-rounded` 类时，页面背景改为透明，用于配合 Electron 无边框透明窗口实现外轮廓圆角。
- `global.css` 中的浮动导航样式已升级：底栏容器改为更轻的磨砂胶囊，激活项采用内嵌高亮、顶部短光条和更克制的 hover/focus 态，用于强化桌面端“量化控制台”观感。
- `global.css` 中的浮动导航已提升与浅色背景的分离度：底栏改为高不透明冷灰玻璃底、增加外描边和双层投影；激活项保留浅色磨砂高亮与蓝色图标文字，避免蓝色实底过重。
- `global.css` 中的浮动导航已试验半透明玻璃版本：底栏采用更低不透明度、高模糊、斜向高光、内高光与边缘暗部，图标使用高光/暗部多向投影和略粗描边模拟浮雕线性效果。
- `global.css` 中的浮动导航现支持居中悬浮胶囊布局：容器限制最大宽度、内容可横向滚动隐藏滚动条、内部按组分段展示，优先降低长导航在桌面端的横向压迫感。
- `global.css` 新增控制台头部与窗口栏样式：顶部状态卡、品牌中枢、时钟侧栏以及窗口控制按钮统一采用浅色玻璃材质、细边框和柔和阴影，以与底部悬浮胶囊形成一致的视觉系统。
